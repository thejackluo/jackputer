RAM64 max

Hierarchical max view. RAM64 expands to 8 RAM8 chips, 64 registers, and 1,024 one-bit storage cells.

A flat DigitalJS render at this depth would create too many browser nodes and wires to be useful. This page keeps the full expansion navigable as a memory hierarchy instead of generating one enormous canvas.
